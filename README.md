## Getting Started

### Clone this repository

```bash
$ git clone https://github.com/TracK92/hello-react-front-end.git
$ cd hello-react-front-end
```

### Run project

```bash
$ npm start

```

### Open page in browser

```bash
$ http://localhost:3000/
```
